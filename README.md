# Blog App

> The Blog app is a classic example of a blog website. A fully functional website that shows the list of posts and empower readers to interact with them by adding comments and liking posts.

## Built With

- Ruby on Rails
- Ruby

## Prerequisites

- Ruby
- Ruby on Rails
- PostgreSQL

## Install

- VScode
- Ruby
- Ruby on Rails(as a gem)
- Postgresql database

## Getting Started

To get a local copy of the app, run the following commands in your terminal:
- `git clone https://github.com/zairdon20/blog-app.git`
- `cd blog-app`
- `bundle install`

Run the following commands to run the app

- `rails db:create`
- `rails db:migrate`
- `rails s`

In your browser, go to

- `http://localhost:3000`

## Testing
- To install rspec: gem install rspec
- To run all the tests: rspec ./spec
